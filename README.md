# Database Design Roadmap

Ruta completa para aprender diseño de bases de datos desde nivel principiante hasta nivel arquitecto de datos.

---

## Objetivos

- Comprender cómo modelar datos.
- Diseñar bases de datos escalables.
- Entender relaciones entre entidades.
- Aplicar normalización correctamente.
- Diseñar sistemas OLTP.
- Diseñar sistemas OLAP.
- Construir Data Warehouses profesionales.
